Installation
============

Install with `pip`
""""""""""""""""""

.. code-block:: bash

  pip install coinmetrics

Manual Install
""""""""""""""

.. code-block:: bash

  git clone https://github.com/h4110w33n/coinmetrics
  cd coinmetrics
  python setup.py install

Now that you have everything installed, you can refer to the :ref:`usage` page for some code snippets.
